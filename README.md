# SmartSchool CC校友录V8

#### 介绍

CC校友录V8(毕业入世版) 
主要面向毕业后在某城市工作的校友，给大家构建一个充分交流的平台，“人脉”积累是本软件的功能特色，
为此淡化了班级，所有的朋友畅所欲言，新老校友信息共享，从不认识到认识，再到熟悉，
通过校友身份这个共同的纽带，为自己的人生道路积累持续的人脉。

CC校友录V8(毕业入世版)是一个采用 ASP 和 Access或SQLServer数据库构建的高效的校友录
解决方案。 CC校友录V8(毕业入世版)在代码质量，运行效率，
